# DAWAA Events - Permissions Update

تحديث على نفس مشروع HTML/CSS/JS:

- حذف النصوص التجريبية من صفحة تسجيل الدخول.
- حذف زر إصلاح البيانات التجريبية وبيانات الدخول الظاهرة للزائر.
- تعديل عبارة الصفحة الرئيسية في الإدارة لتكون عملية أكثر.
- إضافة محرر صلاحيات للعميل داخل صفحة الحسابات.
- يمكن اختيار ما يظهر للعميل: بيانات المناسبة، الضيوف، الحالات، الرسائل، البطاقات، التقرير، التصدير، وطلب إعادة الإرسال.
- بوابة العميل تراعي الصلاحيات المختارة وتخفي الأقسام غير المسموحة.

## التشغيل
افتح `index.html` مباشرة أو ارفعه على Vercel.
